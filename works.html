IOT BASED FOOT PRESSURE MONITORING SYSTEM USING ML ALGORITHMS

This project consist of special sensor based shoe for diabetic patient that collects the flex sensor data and transmit to the server for observation. The collected data is passed to the machine learning based trained model for predicting the foot state. Machine learning algorithms has been implemented to predict the foot pressure state and generate the instruction. We have categorized the states into three states like “normal, moderate and severe”. The collected data is monitored continuously and the states for the foot can be checked in the android based mobile app. .
